## 3. Long Description (Flippa / Acquire format)

```
## What is NutriAID?

NutriAID is a complete, production-ready SaaS platform that helps users 
identify food intolerances through intelligent AI correlation of meals and 
symptoms. It combines a proprietary multi-worker AI orchestrator with a 
rule-based fallback engine, a full admin console, and recurring Stripe 
billing — all in one codebase.

## The Problem It Solves

1 in 4 Europeans suffers from undiagnosed food intolerances. Medical 
diagnosis costs €300–1,500. A dietitian costs €50–150/session. NutriAID 
delivers comparable insights at €10–36/month, using the user's own daily 
data and AI pattern recognition.

## Technical Stack

- **Frontend:** Next.js 14 (App Router), TypeScript, Tailwind CSS, Stripe
- **Backend:** Next.js 15, custom AI orchestrator, JSON data store
- **Database:** PostgreSQL 16 with Drizzle ORM
- **AI:** GPT-4o primary → Gemini 1.5 Pro fallback → Rule-based engine
- **Infra:** Docker + Traefik (HTTPS, TLS, load balancing)

## AI Architecture (Unique)

The platform features an 11-worker AI orchestrator:
- profile-analyzer, intolerance-checker, allergy-checker
- meal-plan-generator, recipe-builder, nutrition-calculator  
- supplement-advisor, progress-tracker, medical-safety
- shopping-list-generator + automated correction worker

Each worker output is schema-validated, semantically validated 
(no medical diagnoses, no absolute language), auto-corrected if invalid 
(GPT-4o → Gemini → rule-based chain), and fully logged.

## Self-Healing Layer

If GPT-4o fails: automatic Gemini 1.5 Pro fallback.
If Gemini fails: rule-based engine (zero API cost).
If everything fails: cached results served.
No human intervention required.

## Admin Console

Complete, zero-code admin panel:
- AI model configuration (any OpenAI-compatible provider)
- Per-worker prompt customization
- Stripe products and prices setup
- Email SMTP configuration
- User management (view, edit, plan assignment)
- Pricing page configuration (no code changes needed)
- PWA settings (app name, colors, push notifications)
- 2FA enforcement for admins
- Full audit trail + orchestrator logs

## Monetization (Ready)

- 7-day free Pro+ trial (no card required)

## GDPR & Legal

✅ GDPR data export (JSON + PDF) on request

✅ Medical safety validator (no diagnoses, no prescriptions in AI output)

- Full IP transfer
```
